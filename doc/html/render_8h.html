Detailed Description

Fonctions de rendu ASCII pour Pixel Tracer.

Ce module gère :

  • l'effacement de l'écran
  • le rendu des zones (Area)
  • le dessin des layers et des shapes

Il constitue la couche de rendu de l'application.
