Detailed Description

Gestion globale des identifiants uniques (ID) utilisés par Pixel Tracer.

Ce module maintient un compteur global d'ID et fournit des fonctions pour obtenir un nouvel ID, définir manuellement l'ID courant, et sauvegarder / charger l'ID depuis un fichier.
