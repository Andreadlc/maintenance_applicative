Detailed Description

Gestion des commandes utilisateur (CLI) pour Pixel Tracer.

Ce module permet de lire les commandes depuis stdin, de parser les paramètres (int, float, string) et d'exécuter les actions correspondantes (dessin, gestion des layers, etc.).
